BeeGrid

A fantastic CSS microframework to build various types of grids thanks to a twelve column system, five default responsive tiers, Sass variables and mixins, and dozens of predefined classes to organize your content.

Installation

Install Beegrid's source files via npm. We take care of injecting the CSS needed.

{% highlight sh %} npm install beegrid {% endhighlight %} {% highlight sh %} npm install beegrid -v 1.0.0 {% endhighlight %}
